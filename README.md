Privacy Policy

AI AppStore Preview
Last updated: May 2026

Overview
AI AppStore Preview is a macOS utility for capturing, editing, and exporting App Store screenshots and video previews. Your privacy is straightforward with this app — it collects nothing and sends nothing anywhere.

Data Collection
AI AppStore Preview does not collect, transmit, store remotely, or share any personal data, usage data, analytics, or telemetry of any kind.

What Stays on Your Device
All data created or used by this app remains exclusively on your Mac:
• Screen captures and video recordings are saved to a local folder you choose (default: ~/​Pictures​/​App​Store​Preview/)
• Your media library index is stored locally at ~/​Library​/​Application ​Support​/​AI​_​App​Store​Preview​/library​.json
• Your storage folder preference is saved in your local User​Defaults
• No data leaves your device at any time

Permissions Used
| Permission | Why |
|---|---|
| Screen Recording | Required to capture screenshots and record video of your screen and app windows |
| File Access (user-selected) | Required to read imported media files and write exports to your chosen output folder |

Third-Party Services
None. The app makes no network requests and uses no third-party SDKs, analytics frameworks, or advertising libraries.

Children's Privacy
This app does not collect data from anyone, including children under 13.

Contact
For questions about this privacy policy, open an issue at the app's repository or contact the developer directly through the App Store listing.

Feature List

Screen Capture  
• Full display screenshot capture with live preview  
• Specific window capture — pick any open window from a list  
• Front window auto-capture — set a countdown, Cmd+Tab to your target, fires automatically  
• Region crop selector — drag to define an exact capture area on screen  
• Countdown timer (3s, 5s, 10s) so you can switch to the target before capture fires  
• Capture survives tab switching — countdown continues even if you navigate away  
• Screen recording (full display or specific window) exported as .mov  
• On-screen capture toast notification visible system-wide  

Media Library  
• Persistent library — media survives app restarts, stored in a user-chosen local folder  
• Import screenshots (PNG, JPEG) and videos via file picker or drag and drop  
• In-app crop editor with 8-handle drag interface, rule-of-thirds grid overlay  
• Open in Preview (or any external editor) with auto-refresh on save  
• File watcher — edits saved externally are reflected immediately in the library  
• Reveal in Finder for any media item  
• Clean Up Files — moves unreferenced files in the storage folder to Trash  
• Remove individual items or clear the entire library  

Export 
• Export screenshots to exact App Store required pixel dimensions  
• Export video previews resized to App Store required resolutions  
• Three scaling modes: Fit (letterbox), Fill/Crop (aspect fill), Stretch  
• Supports all current App Store device targets:  
   • iPhone (6.9", 6.7", 6.5", 5.5")  
   • iPad Pro (13", 12.9", 11")  
   • Mac (App Store screenshots)  
   • Apple Watch  
   • Apple TV  
• Editable filename prefix per export job  
• Live export preview showing how the image will look at target dimensions  
• Orientation mismatch warning when source and target orientations differ  
• Duration warning for video previews outside the 15–30 second App Store requirement  
• Parallel export of multiple jobs  
• Save originals option — copies source files alongside exports  
• Reset individual or all failed/completed jobs to re-export  

Device Selector  
• Visual device picker grouped by category (iPhone, iPad, Mac, Watch, TV)  
• Select all / deselect all per category  
• Selected device count shown per category  

General  
• Native macOS SwiftUI app — no Electron, no web views  
• Fully sandboxed — no network access required or used  
• Security-scoped bookmark for persistent access to your chosen media folder  
• iCloud Drive warning if a cloud-synced folder is selected as storage location
